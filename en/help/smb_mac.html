Open File Sharing
1. Click "System Preferences" --> "Sharing". As shown below:

2. Select "File Sharing" from the service box on the left.
3. In the "Shared Folders:" section, you can add or remove folders to share.
4. In the "Users:" section, you can modify the sharing permissions. As shown below:

5. Click "Options..." to open the sharing settings and check "Share files and folders using SMB".
6. In the account box below, you can choose which user to enable the SMB service for. As shown below:


Firewall Settings
If you can't connect, it's possible that the Mac's firewall is blocking the network connection. Please set up the firewall as follows.
1. Click "System Preferences" --> "Security & Privacy" --> "Firewall".
2. Click the lock symbol in the lower left corner to unlock it.
3. You can choose to "Turn Off the Firewall" or click on "Firewall Options...".
4. Be sure not check "Block all incoming connects". As shown below:
