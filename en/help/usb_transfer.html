USB File Transfer is a transfer method based on iTunes file sharing. Before using this feature, please make sure that your computer has Apple's iTunes software installed. If not, you can get it from Apple official Page is free to download. Below we take iTunes 12.8 as an example to illustrate.

1. First connect the iOS device to the computer with the data cable, then open iTunes and find the connected device.

2. Referring to the figure below, select “File Sharing” (a mark) --> “File Center” (b mark) in the Apps.

3. In the "File Center Documents" (c mark), you can drag and drop files from your computer to the area for uploading, or you can drag and drop files from the area to your computer for saving.

Note: The transferred file can be found in the main interface of File Center --> click on the upper left corner to open the sidebar --> "Files from iTunes".
