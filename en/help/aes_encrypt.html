File AES-256 encryption
1. File Center supports AES-256 file encryption, and the encrypted file can be used for sharing, saving, and so on.
2. In "List" mode, click on the lower triangle to see the "AES Encrypt" operation, as shown below:

2. In "Thumbnail" mode, long press on the file you want to encrypt, you can see the "AES Encrypt" operation, as shown below:

3. After clicking "AES Encrypt", enter the password in the box that appears, or use the default default password. (The default password can be set in the settings)
4. After the encryption is completed, an encrypted file with the extension ".encrypted" is generated in the same directory.
