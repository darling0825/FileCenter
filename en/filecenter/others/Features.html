This is a files management tool with powerful functions and simple interface, which integrates local files and cloud files. Functions include photo viewer and editor, video player, music player, PDF viewer, TXT editor and more. It help users easily manage local files and cloud files (files stored on file servers such as SMB/CIFS, ftp, sftp, and WebDAV, and files stored on cloud drives such as OneDrive, GoogleDrive, Box, Dropbox and more).

Feature Introduction

  • Support file servers such as SMB/CIFS, ftp, sftp, WebDAV
  • Support Cloud Drives such as OneDrive, GoogleDrive, Box, Dropbox, and support permissions and sharing management
  • Support AES 256-bit file encryption
  • Support files of most audio and video format play
  • Support easy switch between local and cloud files
  • Support clipboard management
  • Support recycle bin
  • Support independent task management
  • Support built-in album management, integrate photos and files

Local File Management Function

  • Support files browsing by both grid and list
  • Support file searching
  • Support basic operations such as file move, copy, delete, rename, and ZIP compression
  • Support sort by name, size, type, time and folder first
  • Support file mailing, sharing, collecting, importing to iCloud, etc
  • Supports file opening by using the system's built-in opening mode
  • Supports AES 256-bit encryption
  • Support pictures storing to system album
  • Support to create directories, create new text, import from photo albums, import from iCloud, take photos and videos, etc
  • Support system clipboard management, storing content from clipboard to files, etc

Cloud File Management Function

  • Supports file servers such as SMB/CIFS, ftp, sftp, WebDAVD
  • Supports Cloud Drives such as OneDrive, GoogleDrive, Box, Dropbox
  • Support files browsing by both grid and list; same function as local file management
  • Support sort by name, size, type, time and folder first
  • Support quick switching between local and cloud files
  • Support permissions management, file sharing (OneDrive, GoogleDrive, Box, Dropbox)
  • support operations such as upload, download, delete, rename, move, copy, and create directories for Cloud files
  • Support taking photos, videos, and storing them in cloud files
  • Support iCloud files importing and exporting
  • Support collecting directories which include music files into music players
  • Support collecting directories which include video files to video player
  • Support shortcuts for directories
  • Support multiple files download with dedicated task management

File Transfer Method

  • Support WiFi transmission
  • Support iPhone/iPad as WebDAV server
  • Support file transmission from iTunes shared files

Photo Management Function

  • Support deleting photos in the system album directly
  • Support photo clipping, rotation and other editing
  • Support super long and super flat images browsing
  • Support pictures zooming in and out
  • Support gif format play

Music Play Function

  • Support files of most audio formats play
  • Support audio files from server and cloud drives play
  • Support customized music playlist
  • Support adding web playlist
  • Support adding to music playlist from Cloud File Browser
  • Support loop, single, random and other play modes
  • Support operations such as previous song, next song, pause, start, etc
  • Support sidebar mini play window

Video Play Function

  • Support files of most video formats play
  • Support video files from server and cloud drives play
  • Support customizing video playlist
  • Support adding web playlist
  • Support adding to video playlist from Cloud File Browser
  • Support loop, single, random and other play modes
  • Support screen locking to prevent misuse

Clipboard Management Function

  • Built-in light volume clipboard management
  • Manage text, pictures and more from the clipboard
  • Export clipboard contents to files

PDF Browsing Function

  • Support bookmark structure
  • Support brightness adjustment
  • Support content searching
  • Support sharing and collecting etc
